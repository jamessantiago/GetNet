Cross Platform with .NET Core

GetNet is built on top of .NET Core, Entity Framework Core, and the Kestrel web server. This means that GetNet can run on multiple operation systems such as Windows and Linux; use different database backends such as Microsoft SQL and Postgres; and run behind different full featured web servers such as IIS, Apache, and Nginx. If desired, GetNet can run on no-cost-licensed infrastructure.
